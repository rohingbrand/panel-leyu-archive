# panel-leyu-archive

## 项目简介

`panel-leyu-archive` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定网站、域名或应用，仅作为静态页面集合的存储与分发空间，方便用户查阅、浏览或引用历史归档内容。

## 目录结构

```
panel-leyu-archive/
├── index.html          # 主入口页面，提供归档索引
├── pages/              # 存放所有归档的 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/             # 静态资源（如图片、样式表、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下为各个独立的 HTML 归档页面，文件名遵循统一命名规范。
- `assets/` 目录存放页面所需的共用资源，避免重复引用。

## 页面归档说明

- 所有归档页面均为独立的 HTML 文件，可直接在浏览器中打开查看。
- 页面内容基于历史数据整理，仅供存档和参考使用。
- 归档页面不包含任何动态后端依赖，完全静态化。
- 文件名以 `page-` 开头，后接数字编号，便于索引和排序。
